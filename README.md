Project Overview: Developed a terminal-based Volunteer Management System using Python to streamline internal operations for NayePankh Foundation.

Core Administration: Built an interactive command-line interface that allows administrators to easily register new volunteers with custom details.

Data Fields Tracked: Captures critical volunteer tracking information including Name, Email, Phone Number, City, and operational domains.

Information Retrieval: Implemented a data view module that formats and prints all active volunteer records directly to the console screen.

Search Functionality: Programmed a case-insensitive search engine to quickly look up individual profiles by typing in a volunteer's name.

Persistent File Handling: Integrated Python's built-in csv module to handle background data storage and keep user files organized.

Data Security & Safety: Configured the application to permanently append new entries to a local volunteers.csv sheet so no data is ever lost.

Automated Metrics Reporting: Designed a report generation tool that loops through data to instantly calculate the total number of registrations.

Domain Categorization: Programmed the reporting engine to count and display real-time breakdowns of volunteers sorted by their chosen field of interest.

Operational Readiness: Delivered a clean, modular script structure utilizing loops, functions, and file-handling concepts suitable for scaling.
